# golookup: a very simple DNS lookup tool

golookup gets the ip addresses of a domain name

## usage

```sh
golookup google.com
```

## installation

### with go

```sh
go install github.com/leanghok120/golookup
```

### build from source

```sh
git clone https://github.com/leanghok120/golookup.git
cd golookup
go install
```
